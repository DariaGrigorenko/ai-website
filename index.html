Поэтапное создание сайта с помощью ИИ

HGGps создаёт сайт по шагам: сначала читает описание проекта, затем помогает настроить фон, разместить кнопки и блоки, добавить информацию и получить ссылку на финальный сайт.

Конструктор сайта

Заполняй этапы по очереди и смотри, как сайт собирается постепенно.

1. Описание
2. Фон
3. Блоки
4. Информация
5. Ссылка

Шаг 1. Описание проекта

Сначала сайт читает описание проекта и понимает, какой сайт нужно создать.

Шаг 2. Настройка фона

Теперь пользователь описывает, каким должен быть фон сайта, и оценивает результат.

Шаг 3. Размещение кнопок и блоков

Пользователь выбирает, где будут находиться основные блоки и кнопки.

Шаг 4. Добавление информации

Теперь пользователь добавляет информацию в нужные части сайта.

Шаг 5. Финальный сайт и ссылка

Все этапы завершены. Теперь можно создать публичную ссылку через backend.

Предпросмотр

Заполни первый этап, чтобы начать сборку сайта.
Текущий этап: ожидание описания проекта.
