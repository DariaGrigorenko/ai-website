# ai-website

ИИ-конструктор веб-сайтов, который генерирует структуру, тексты и дизайн лендинга по текстовому описанию пользователя.

## Описание

Пользователь вводит описание бизнеса, выбирает цель сайта и стиль, после чего система создаёт готовый лендинг. В MVP пользователь может посмотреть результат, отредактировать основные тексты и опубликовать сайт по ссылке.
