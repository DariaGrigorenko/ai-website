**От идеи до сайта за минуты. \\  From idea to website in minutes.**

ИИ-конструктор веб-сайтов, который генерирует структуру, тексты и дизайн лендинга по текстовому описанию пользователя.

## Описание

Пользователь вводит описание бизнеса, выбирает цель сайта и стиль, после чего система создаёт готовый лендинг. В MVP пользователь может посмотреть результат, отредактировать основные тексты и опубликовать сайт по ссылке. Сервис поддерживает создание простых сайтов: лендингов и многостраничных сайтов до 5 страниц.

## Возможности MVP

В MVP входит:
 - создание сайта по текстовому описанию;
 - выбор типа сайта;
 - выбор цели сайта;
 - выбор стиля сайта;
 - генерация карты сайта;
 - генерация страниц сайта;
 - генерация текстов;
 - предпросмотр сайта;
 - переключение между страницами;
 - редактирование основных текстов;
 - публикация сайта по ссылке.

## Ограничения MVP

В MVP не входит:
 - интернет-магазин;
 - онлайн-оплата;
 - личный домен;
 - сложный drag-and-drop редактор;
 - личный кабинет клиента;
 - CRM-интеграции;
 - аналитика посещений;
 - генерация изображений;
 - экспорт в HTML/CSS/React.
