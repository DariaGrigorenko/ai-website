# HGGps — ИИ-конструктор сайтов

Проект состоит из frontend на GitHub Pages и backend на Render.

Новая логика:
- пользователь выбирает только тип сайта: лендинг или многостраничный сайт;
- пользователь описывает проект, оформление, нужную информацию, контакты и количество кнопок;
- Gemini сам выбирает фон, стиль, карту сайта, расположение блоков, тексты и кнопки;
- если телефон или почта не указаны, backend подставляет резервные значения;
- после генерации пользователь видит предпросмотр сайта;
- если сайт не понравился, его можно перегенерировать.

## Render

Root Directory:

```text
backend
```

Build Command:

```text
pip install -r requirements.txt
```

Start Command:

```text
uvicorn main:app --host 0.0.0.0 --port $PORT
```

Environment Variables:

```text
GEMINI_API_KEY=твой_ключ
GEMINI_MODEL=gemini-2.5-flash
```

## Frontend

В `script.js` должен быть адрес Render:

```js
const API_URL = "https://ai-website-exwx.onrender.com";
```


## Изображения

Сгенерированные сайты теперь получают тематическое изображение. ИИ выбирает смысловую категорию визуальные блоки, а backend подставляет подходящую картинку из заранее подготовленной библиотеки открытых изображений.
